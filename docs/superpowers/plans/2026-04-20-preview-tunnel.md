# Preview Tunnel Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Expose a local `hugo server` at `preview.chapter23.uk` via a named cloudflared tunnel, with the tunnel auto-starting on login.

**Architecture:** Create a named cloudflared tunnel (`chapter23-preview`) with its own yml config file (matching the mnemo-dashboard.yml pattern already in use). A user LaunchAgent keeps the tunnel running on login. Hugo server is started manually in a tmux session when working on the site. A helper script `~/bin/site-preview` handles Hugo startup/teardown.

**Tech Stack:** cloudflared 2026.3.0, Hugo, tmux, macOS LaunchAgent

### Task 2: Write the tunnel config

**Files:**
- Create: `~/.cloudflared/chapter23-preview.yml`

- [ ] **Step 1: Get the tunnel UUID**

```bash
cloudflared tunnel list | grep chapter23-preview
```

Copy the UUID from the output.

- [ ] **Step 2: Write the config file**

Replace `<UUID>` with the actual tunnel UUID from Step 1:

```bash
cat > ~/.cloudflared/chapter23-preview.yml << 'EOF'
tunnel: <UUID>
credentials-file: /Users/chapter23/.cloudflared/<UUID>.json

ingress:
  - hostname: preview.chapter23.uk
    service: http://127.0.0.1:1313
  - service: http_status:404
EOF
```

- [ ] **Step 3: Verify the config**

```bash
cat ~/.cloudflared/chapter23-preview.yml
```

Confirm `tunnel:` shows the UUID and `credentials-file:` path exists:

```bash
ls ~/.cloudflared/<UUID>.json
```

### Task 5: Create the site-preview helper script

**Files:**
- Create: `~/bin/site-preview`

- [ ] **Step 1: Write the script**

```bash
cat > ~/bin/site-preview << 'EOF'
#!/bin/bash
# Start or stop the Hugo preview server in a tmux session

SESSION="hugo-preview"
SITE_DIR="$HOME/dev/chapter23.uk"

case "${1:-start}" in
  start)
    if tmux has-session -t "$SESSION" 2>/dev/null; then
      echo "Preview already running. Visit https://preview.chapter23.uk"
      echo "Attach with: tmux attach -t $SESSION"
    else
      tmux new-session -d -s "$SESSION" -c "$SITE_DIR" \
        "hugo server --bind 127.0.0.1 --baseURL https://preview.chapter23.uk --disableFastRender"
      echo "Preview started. Visit https://preview.chapter23.uk"
      echo "Attach with: tmux attach -t $SESSION"
      echo "Stop with:   site-preview stop"
    fi
    ;;
  stop)
    tmux kill-session -t "$SESSION" 2>/dev/null && echo "Preview stopped." || echo "Preview was not running."
    ;;
  status)
    if tmux has-session -t "$SESSION" 2>/dev/null; then
      echo "Running. Visit https://preview.chapter23.uk"
    else
      echo "Not running. Start with: site-preview"
    fi
    ;;
  *)
    echo "Usage: site-preview [start|stop|status]"
    ;;
esac
EOF
```

- [ ] **Step 2: Make it executable**

```bash
chmod +x ~/bin/site-preview
```

- [ ] **Step 3: Verify it's on PATH**

```bash
which site-preview
```

Expected: `/Users/chapter23/bin/site-preview`
